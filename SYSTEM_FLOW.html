Authentication Flow
Login, Register, JWT refresh, role-based guards, and protected routes.

🔐 Login & Token Flow AUTH

sequenceDiagram participant B as Browser participant FE as Next.js Frontend participant BE as NestJS Auth Module participant DB as MongoDB B->>FE: Navigate to /auth/login FE->>B: Render LoginPage B->>FE: Submit email + password FE->>BE: POST /auth/login { email, password } BE->>DB: Find user by email DB-->>BE: User document BE->>BE: bcrypt.compare(password, hash) alt Valid credentials BE->>BE: Generate JWT (access + refresh) BE-->>FE: { user, accessToken, refreshToken } FE->>FE: Store tokens in httpOnly cookie / context FE->>B: Redirect to role-based dashboard else Invalid credentials BE-->>FE: 401 Unauthorized FE->>B: Show error toast end Note over FE,BE: Protected routes send Authorization: Bearer {token} B->>FE: Access protected page FE->>BE: GET /auth/me (Bearer token) BE->>BE: Validate JWT alt Token valid BE-->>FE: Current user data FE->>B: Render protected page else Token expired FE->>BE: POST /auth/refresh (refreshToken) BE-->>FE: New accessToken FE->>BE: Retry original request end
